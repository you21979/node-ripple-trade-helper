node-ripple-trade-helper
========================
ripple上の取引システムを簡単に扱えるようにしたライブラリ

rippleのシステムは扱える商品が複雑なので取引所という単位でドメインを絞るようにした

クラスの初期化時に取引所情報に相当するissuerとwalletを指定する

* issuer
* wallet

この情報に対して操作を行うライブラリを提供する

exchange
--------

* buy
* sell
* orderlist
* cancel
* orderbook

wallet
------

* balance
* withdraw
